# FarmHealth AI - Team Roles & Responsibilities

## Project Overview

FarmHealth AI is a fully offline agricultural and livestock-health advisor designed for African farmers and extension officers.

The solution is being developed for the Africa Deep Tech Challenge (ADTC) 2026 and is optimized to run on low-resource laptops without internet connectivity.

# Project Goal

Build an accurate, efficient, and fully offline agricultural intelligence system capable of providing crop, livestock, livestock-health, and farm decision-support guidance on commodity 8GB laptops.
